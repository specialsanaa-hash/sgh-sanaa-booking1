## 🛠️ التقنيات المستخدمة

### Frontend
- **React 19** - مكتبة واجهات المستخدم
- **TypeScript** - لغة البرمجة
- **Tailwind CSS 4** - تصميم الواجهات
- **shadcn/ui** - مكونات واجهة جاهزة
- **tRPC** - اتصال آمن بـ Backend
- **Socket.io Client** - اتصالات فورية
- **Wouter** - نظام التوجيه

### Backend
- **Express 4** - خادم ويب
- **tRPC 11** - API آمن
- **Drizzle ORM** - إدارة قاعدة البيانات
- **MySQL/TiDB** - قاعدة البيانات
- **Zod** - التحقق من صحة البيانات
- **Socket.io** - اتصالات فورية

### Authentication
- **Manus OAuth** - نظام المصادقة المدمج
- **JWT** - توقيع الجلسات
- **Session Cookies** - إدارة الجلسات
