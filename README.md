# منصة حجز المستشفى السعودي الألماني - صنعاء

منصة إلكترونية متكاملة لاستقبال حجوزات المستشفى السعودي الألماني - صنعاء وإدارة بيانات المرضى من خلال حملات السوشيال ميديا (Meta/Facebook).

## المميزات الرئيسية

### 🏥 صفحات الهبوط والحجز
- صفحة هبوط احترافية تعرض معلومات المستشفى والشعار الرسمي
- نماذج حجز ديناميكية قابلة للتخصيص حسب كل حملة إعلانية
- تجربة مستخدم سلسة مع رسائل تأكيد فورية
- دعم كامل للغة العربية

### 📊 لوحة التحكم الشاملة
- لوحة تحكم متقدمة للمسؤولين لإدارة الحملات والحجوزات
- عرض إحصائيات الحجوزات والحملات في الوقت الفعلي
- قائمة كاملة بالحجوزات مع إمكانية البحث والتصفية
- عرض تفاصيل كل حجز

### 🎯 إدارة النماذج المخصصة
- واجهة متقدمة لإنشاء نماذج مخصصة جديدة
- إضافة حقول ديناميكية متعددة الأنواع:
  - نص عادي
  - بريد إلكتروني
  - رقم هاتف
  - أرقام
  - تاريخ
  - قائمة اختيار
  - نص طويل
- حذف وتحرير الحقول بسهولة
- ربط النماذج بالحملات الإعلانية

### 📥 التصدير والتقارير
- تصدير الحجوزات إلى ملف Excel مع معاينة البيانات
- تصفية متقدمة حسب:
  - التاريخ (من - إلى)
  - حالة الحجز (قيد الانتظار، مؤكد، ملغى، مكتمل)
  - الحملة الإعلانية
- واجهة سهلة الاستخدام للتصدير

### 🔐 الأمان والمصادقة
- نظام مصادقة آمن مع Manus OAuth
- نظام الأدوار (Admin/User)
- حماية من الهجمات الشائعة (CSRF, XSS)
- تسجيل الأنشطة والعمليات

## البنية التقنية

### الواجهة الأمامية
- **React 19** مع TypeScript

- **Express 4** لخادم الويب

### قاعدة البيانات
- **MySQL** لتخزين البيانات
- **Drizzle ORM** لإدارة قاعدة البيانات
- جداول محسّنة للأداء والأمان

### الخدمات الإضافية
- **S3 Storage** لتخزين الملفات والوسائط
- **Manus OAuth** للمصادقة
- **Global CDN** لتوزيع المحتوى بسرعة

## 🔌 تكامل Socket.io مع تطبيق بوابة الرسائل

المنصة مدمجة مع نظام Socket.io المتقدم لإرسال رسائل SMS و WhatsApp تلقائية إلى المرضى.

### رابط الاتصال
```
wss://sghsanaa-ba99upcz.manus.space/socket.io/?apiKey=sk_b099b0f8f4860da58325ac2e8860e24057ccf5b588108acf9a8acf3a4e7955c4
```

### الأحداث المدعومة
- **send_message** - إرسال رسائل SMS/WhatsApp
- **message_response** - تقرير حالة الرسالة (sent/failed/pending)
- **device_status** - تحديثات حالة الجهاز (البطارية، الشبكة)
- **command** - إرسال أوامر للتطبيق
- **command_response** - استجابة التطبيق للأوامر

### جداول قاعدة البيانات
- `apiKeys` - إدارة مفاتيح API
- `socketConnections` - تتبع الاتصالات النشطة
- `socketMessageLogs` - سجل جميع الرسائل والأحداث

## الإجراءات المتاحة (tRPC)

### الحملات (Campaigns)
- `campaigns.list` - عرض جميع الحملات
- `campaigns.create` - إنشاء حملة جديدة
- `campaigns.getById` - الحصول على تفاصيل حملة

### النماذج (Forms)
- `forms.list` - عرض جميع النماذج
- `forms.create` - إنشاء نموذج جديد
- `forms.getById` - الحصول على تفاصيل نموذج

### الحقول (FormFields)
- `formFields.list` - عرض حقول النموذج
- `formFields.create` - إضافة حقل جديد
- `formFields.delete` - حذف حقل

### الحجوزات (Bookings)
- `bookings.list` - عرض جميع الحجوزات
- `bookings.create` - إنشاء حجز جديد
- `bookings.getById` - الحصول على تفاصيل حجز

### إجابات النماذج (FormResponses)
- `formResponses.list` - عرض الإجابات
- `formResponses.create` - إضافة إجابة جديدة
